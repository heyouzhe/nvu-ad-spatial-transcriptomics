# NVU AD Spatial Transcriptomics

This repository will host code and analysis notes for a neurovascular unit
(NVU) project studying Alzheimer's disease with single-cell spatial
transcriptomics.

## Project Overview

The project focuses on NVU dysregulation across the hippocampus and prefrontal
cortex in Alzheimer's disease. It aims to reconstruct spatially resolved NVUs,
characterize disease-associated molecular and cellular changes, and integrate
multi-scale features with graph neural network models.

## Main Questions

- How are NVUs spatially reorganized in Alzheimer's disease?
- Which hippocampal and cortical regions show the strongest NVU alterations?
- How do vascular, immune, glial, and ligand-receptor communication features
  change between AD and control samples?
- Can graph neural network models integrate NVU-level features to classify AD
  and rank regional or molecular vulnerability?

## Planned Contents

- `data/`: data access notes, metadata templates, and preprocessing inputs
- `notebooks/`: exploratory analyses and figure checks
- `scripts/`: reusable analysis scripts
- `results/`: processed outputs, tables, and model summaries
- `docs/`: project notes and manuscript-related documentation

## Current Status

This repository is intentionally code-empty for now. The initial purpose is to
provide a concise project introduction and a clean place to add analysis code
later.
